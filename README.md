Installation:
1. Clone the repo:
   ```bash
   git clone https://github.com/InfinitelyFinite/x-clone.git
   cd x-clone
   ```
2. Install dependencies:
   - Frontend:
     ```bash
     cd frontend
     npm install
     ```
   - Backend:
     ```bash
     cd ../backend
     npm install
     ```
3. Start development servers:
   - Frontend (port 3000):
     ```bash
     cd frontend
     npm run dev
     ```
   - Backend (port 5000):
     ```bash
     cd ../backend
     npm run dev
     ```
4. View in browser:
   ```bash
   open http://localhost:3000
   ```
